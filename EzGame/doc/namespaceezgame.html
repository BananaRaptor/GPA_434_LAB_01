◆ Alignment

enum class ezgame::Alignment
strong

Définit l'alignement des objets graphiques.

Cette énumération est utilisée pour aligner les objets de type Circle et Text lors de l'affichage. La position finale sur la surface graphique dépend :

  • la position
  • la taille
  • l'alignement

Les fonctions Screen::draw (2 surcharges) utilisent toutes ces informations pour déterminer la position finale lors de l'affichage.
