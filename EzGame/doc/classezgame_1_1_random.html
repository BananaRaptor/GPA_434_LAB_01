Description détaillée

Classe statique générant des valeurs aléatoires de divers types.

Cette classe offre une collection de méthodes statiques permettant de
générer des valeurs aléatoires. Elle est conçue pour être purement statique et ne peut donc pas être instanciée.

Les types pouvant être générés sont :

  • booléen bool
  • entier int (tous les entiers avec leurs variantes de taille et de signe)
  • réel float (toutes les tailles de points flottant)
  • enumération enum (un enum personnalisé)

Les valeurs générées le sont de façon pseudo aléatoire avec une distribution uniforme.

Génère un énumérateur parmi ceux donnés.

Cette fonction est un template déductible. Ainsi, il n'est pas nécessaire de fournir le type souhaité. Toutefois, il est essentiel que tous les énumérateurs soient du même type énuméré.

Paramètres
firstEnumeratorUn émumérateur.
allOtherEnumeratorsUn nombre variable d'arguments du même type que firstEnumerator.
Renvoie
L'énumérateur généré aléatoirement parmi tous ceux passés en argument.

Exemple d'utilisation :

// un SwitchState aléatoire
enum class SwitchState { Off, On };
SwitchState state{ Random::enumerator(SwitchState::Off, SwitchState::On) };
// un AccessLevel aléatoire
enum class AccessLevel { Admin, User, Guest };
AccessLevel level{ Random::enumerator(AccessLevel::Admin, AccessLevel::User, AccessLevel::Guest) };
// un jour ouvrable aléatoire parmi mercredi, jeudi ou vendredi uniquement
enum class WorkDay { Monday, Tuesday, Wednesday, Thursday, Friday };
WorkDay workDay{ Random::enumerator(WorkDay::Wednesday, WorkDay::Thursday, WorkDay::Friday) };

Génère un énumérateur.

Cette fonction est un template déductible. Ainsi, il n'est pas nécessaire de fournir le type souhaité.

Cette fonction ne fonctionne que pour les énumérations où tous les énumérateurs sont défini entre 0 et n - 1.

Paramètres
lastEnumeratorLe dernier énumérateur défini.
lastIsCountEnumeratorDétermine sur le dernier énumérateur n'est qu'un énumérateur technique de compte (généralement appelé _count_ ou similairement).
Renvoie
L'énumérateur généré aléatoirement parmi tous ceux passés en argument.

Exemple d'utilisation :

// un SwitchState aléatoire
enum class SwitchState { Off, On };
SwitchState state{ Random::enumerator(SwitchState::On) };
// un AccessLevel aléatoire
enum class AccessLevel { Admin, User, Guest, __count__ };
AccessLevel level{ Random::enumerator(AccessLevel::__count__, true) };

Génère un booléen selon la probabilité donnée.

La probabilité corresponda au pourcentage d'obtenir la valeur vraie. Cette valeur est limitée dans l'intervalle [0, 1].

Paramètres
probabilityLa probabilité pour laquelle l'évènement est vrai. Correspond à un pourcentage déterminé dans l'intervalle [0, 1].
Renvoie
Un booléen généré aléatoirement.

Exemple d'utilisation :

// un booléen aléatoire avec une probabilité de 50% faux et 50% vrai
bool myFirstEvent{ Random::event() };
// un booléen aléatoire avec une probabilité de 25% faux et 75% vrai
bool mySecondEvent{ Random::event(0.75f) };

Génère un entier selon la plage totale existante pour le type spécifié.

Cette fonction est un template non déductible. Ainsi, il est obligatoire de fournir le type souhaité.

Renvoie
La valeur entière générée aléatoirement.

Exemple d'utilisation :

// un nombre aléatoire dans l'intervalle :
// - [-2^31, 2^31 - 1]
// - [-2'147'483'648, 2'147'483'647]
int value0{ Random::integer<int>() };
// un nombre aléatoire dans l'intervalle :
// - [0, 2^64]
// - [0, 18'446'744'073'709'551'615]
size_t value1{ Random::integer<size_t>() };
