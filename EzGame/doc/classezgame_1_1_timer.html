Description détaillée

Classe utilitaire donnant accès à des métriques de temps écoulé.

Cette classe donne un accès facile :

  • au temps écoulé :
    • depuis le démarrage de l'application
    • depuis le dernier tic
  • en :
    • milliseconde (un nombre entier de 64 bits)
    • seconde (un nombre réel de 32 bits)
  • une estimation du nombre de tics par seconde.

Documentation des fonctions membres

◆ fpsEstimation()

float ezgame::Timer::fpsEstimation ( ) const

Retourne une estimation du nombre de tic par seconde moyen (frame per second).

Attention, l'estimation est basée sur une moyenne du temps écoulé entre chacun des n derniers tics. Ainsi, les n premières évaluations sont approximatives et biaisées.

La valeur n est par défaut 1000.

Renvoie
Une estimation du nombre de tic par seconde.

◆ secondSinceLastTic()

float ezgame::Timer::secondSinceLastTic ( ) const

Retourne le temps écoulé depuis le dernier tic en seconde.

Le temps écoulé est calculé entre chaque appel de la fonction processEvents.

Attention, la conversion en nombre réel engendre une perte de précision.

Renvoie
Le temps écoulé depuis le dernier tic (approximation).

◆ secondSinceStartup()

float ezgame::Timer::secondSinceStartup ( ) const

Retourne le temps écoulé depuis le démarrage de l'application en seconde.

Le temps donné est celui depuis la création de l'application (objet Application) et non depuis l'appel de la fonction Application::run.

Attention, la conversion en nombre réel engendre une perte de précision.

Renvoie
Le temps écoulé depuis la création de l'application (approximation).
