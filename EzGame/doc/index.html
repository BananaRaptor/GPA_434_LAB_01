Bibliothèque EzGame

Introduction

La bibliothèque EzGame permet de créer facilement une application graphique rudimentaire.

Ce document contient la documentation de l'API de la bibliothèque EzGame.

Sommaire

La bibliothèque est vraiment simple :

  • crée une application graphique simple
  • écoute le clavier
  • affiche à l'écran un texte ou un cercle
  • tourne en boucle et appelle deux fonctions de contrôle :
    • logique selon le temps écoulé et l'écoute du clavier
    • affichage

La simplicité de la bibliothèque permet de focuser sur l'application elle-même plutôt que le cosmétique ou l'apprentissage d'une grande librairie.

L'idée principale consiste à créer une classe représentant un moteur de jeu selon un patron spécifique. Ensuite, il suffit d'instancier une ezgame::Application et d'appeler la fonction ezgame::Application::run avec la classe développée.

Voici un exemple de code présentant un exemple minimal de l'utilisation de EzGame. La classe développée se nomme GameEngine.

#include <EzGame>
class GameEngine {
public:
GameEngine()
: mText("Ceci est un test!", 36.0f, ezgame::Vect2d(400.0f, 300.0f), ezgame::Color::White, ezgame::Alignment::CenterCenter)
, mCircle(50.0f, ezgame::Vect2d(400.0f, 450.0f), ezgame::Color::Yellow, ezgame::Color::Red, 5.0f, ezgame::Alignment::CenterCenter)
{
}
float width() const { return 800.0f; }
float height() const { return 600.0f; }
std::string title() const { return "Minimal game engine"; }
std::string iconFileName() const { return ""; }
bool provessEvents(ezgame::Keyboard const& keyboard, ezgame::Timer const& timer) {
if (keyboard.isKeyPressed(ezgame::Keyboard::Key::Space)) {
mCircle.move(ezgame::Vect2d::fromRandomized() * 2.5f);
}
return !keyboard.isKeyPressed(ezgame::Keyboard::Key::Escape);
}
void processDisplay(ezgame::Screen& screen) {
screen.clear();
screen.draw(mText);
screen.draw(mCircle);
}
private:
ezgame::Text mText;
ezgame::Circle mCircle;
};
int WinMain()
{
ezgame::Application app;
app.run<GameEngine>();
return 0;
}
