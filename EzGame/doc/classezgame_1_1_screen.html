Description détaillée

Classe permettant de dessiner à l'écran.

La classe Screen permet de dessiner sur la surface graphique les primitives disponibles avec la librairie EzGame.

Il est possible de :

  • connaître la taille de la surface graphique :
    • Screen::width
    • Screen::height
  • remplir la surface graphique d'une couleur uniforme :
    • Screen::clear
  • dessiner un objet Circle ou Text :
    • Screen::draw

void ezgame::Screen::clear ( )

Rempli l'écran de la couleur noire.

La couleur étant opaque, cette fonction a pour effet d'effacer tous les éléments présents.

◆ clear() [2/2]

void ezgame::Screen::clear ( Color const &  color)

Rempli l'écran de la couleur donnée.

Puisque la transparence de la couleur est utilisée :

  • une couleur opaque a pour effet d'effacer tout ce qui se trouve sur la surface graphique
  • une couleur semi transparente permet un effet intéressant

◆ draw() [1/2]

void ezgame::Screen::draw ( Circle const &  circle)

Dessine le cercle donné.

Le cercle est dessiné en considérant :

  • taille et positionnement selon :
    • le rayon
    • la position
    • l'alignement
  • l'apparence selon :
    • la couleur de remplissage
    • la couleur et l'épaisseur du contour

Voir la classe Circle pour plus de détails.

◆ draw() [2/2]

void ezgame::Screen::draw ( Text const &  text)

Dessine le texte donné.

Le texte est dessiné en considérant :

  • taille et positionnement selon :

    • la position
    • l'alignement
  • l'apparence selon :
    • la couleur de remplissage
    • la couleur et l'épaisseur du contour
